🧩 USER STORY — TAX DEPRECIATION CALCULATOR (INDONESIA)
📝 Title

Tax Depreciation & Amortization Calculator (DJP Standard)

👤 User Story

As a business owner or taxpayer
I want to calculate asset depreciation based on Indonesian tax regulations
So that I can accurately report financial data and comply with tax rules

✅ ACCEPTANCE CRITERIA (COMPLIANT WITH DJP)
🎯 Asset Input
User can input:
Asset type:
Tangible (Harta Berwujud)
Intangible (Harta Tak Berwujud)
Asset category:
Non-building
Building (permanent / non-permanent)
Acquisition cost (IDR)
⚙️ Asset Classification
📌 Non-Building Assets
User selects group:
Group	Useful Life	Straight-Line	Declining
1	4 years	25%	50%
2	8 years	12.5%	25%
3	16 years	6.25%	12.5%
4	20 years	5%	10%
🏢 Building Assets
Permanent:
20 years (5%)
Non-permanent:
10 years (10%)
⚙️ Method Selection
User can select:
Straight-Line (default)
Declining Balance
🧮 Calculation Logic
📌 Straight-Line
Depreciation per year:
Cost × rate
📌 Declining Balance
Depreciation:
Book value × rate
Final year:
Remaining value fully depreciated
📊 Result Display
System displays:
Annual depreciation
Book value per year
Total depreciation
📈 Visualization
System displays:
Table (year-by-year breakdown)
Optional chart (value decline)
⏱ Start Time Logic
Default:
Depreciation starts:
👉 month of acquisition
Optional (advanced):
Start when asset is used
Start when asset produces income
🔁 User Interaction
User can:
Select asset group
Change method
Recalculate instantly
Reset form
⚠️ Validation
Cost must be positive
Asset group must be selected
Method must be selected
📌 Informational Content
System explains:
Difference between methods
Meaning of asset groups
Basis (PMK No.72/2023)
❗ Disclaimer (WAJIB BANGET 🔥)
System displays:

Calculation is based on Indonesian tax regulations and is for estimation purposes only.
Please consult a tax professional or refer to official guidelines for reporting.

🧠 BONUS (LEVEL SENIOR 🔥)
💡 Edge Cases
Asset not listed → default Group 3
Switching method mid-way (not allowed → show warning)
Partial year depreciation
💡 UX Improvement
Show:
“Your asset will be fully depreciated in X years”
Highlight:
Remaining book value
💡 Advanced Feature (NEXT LEVEL 🚀)
📅 Monthly depreciation
📊 Multiple assets tracking
📁 Export (Excel/PDF)
🧾 Tax report format
